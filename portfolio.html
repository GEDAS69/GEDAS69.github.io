Mathematical Objects with MathML

Below is an example of a complex integral expression displayed using MathML and MathJax.

The integral shown is commonly found in complex analysis, representing the value of a function f(a) defined by a contour integral around a singularity.

f ( a ) = 1 2 π i ∮ f ( z ) ( z − a ) d z

This formula often relates to residues in the complex plane and can help evaluate integrals via the Residue Theorem.
